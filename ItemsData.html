Icons¶

Big list of items with their ID, Icon ID, type and slot can be found in data/gamedata/itemdata.csv (0x2A87C15DF8425A8C) in data.myp.

Icons images are stored in interface.myp in DirectDraw_Surface format.

Association between ID and image file name are stored in interface/default/eatemplate_icons/source/icons.xml (0xBC8564953C316BF6) in interface.myp.
